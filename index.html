Architect

Architect is a library for scaffolding methods
download .ZIPdownload .TGZ

Architect is a library that makes scaffolding and file manipulation easy. It gives you quite a few methods for doing things like:

  • Creating, copying, & rendering files
  • Creating & copying directories
  • Installing gems, running system commands, interacting with vcs
  • Verifying things worked

Architect's main goal is to become the library of choice for gems like rake and thor.

Using Architect

You can include architect modules into a class like so:

require "architect"

class MakeProjectCommand
  include Architect::Core

  def build(name)
    create_directory name do
      copy_file "Gemfile", version: Rails::VERSION
      create_directory "app" do
        create_directory "models"
        create_directory "controllers"
        create_directory "views"
      end
    end
  end
end

or you can simply have them accessible in main (sinatra style):

require "architect"

puts "What is your project name?"
name = gets.chomp

create_directory name do
  copy_file "Gemfile", version: Rails::VERSION
  create_directory "app" do
    create_directory "models"
    create_directory "controllers"
    create_directory "views"
  end
end

The Gemfile file would look like this:

# source/Gemfile
source "https://rubygems.org"

gem "rails", "~> <%= version %>"

We run Erubius over each project which you can provide data to if you pass in a Hash or any object that responds to methods called in the file.

Installing Architect

Add this line to your application's Gemfile:

gem 'architect'

And then execute:

$ bundle

Or install it yourself as:

$ gem install architect
